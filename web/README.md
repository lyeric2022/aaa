## Live Judge/Coach Agent Panel

The move detail page calls the Python bridge at `JUDGE_BRIDGE_URL` (default
`http://localhost:8010`). Start the real uAgents first:

```bash
cd ../agents
python coach_agent.py  # copy printed address into COACH_ADDRESS
python judge_agent.py  # copy printed address into JUDGE_ADDRESS
uvicorn web_bridge:app --port 8010 --reload
```

With `BRIDGE_MODE=agent`, the bridge sends a structured uAgent request to the
running Judge, and Judge asks Coach for fixes when needed.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
